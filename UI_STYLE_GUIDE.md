# Campus Memory UI 风格指南

基于 X (Twitter) 平台 2024-2025 UI 设计风格，为 Campus Memory 项目定制的视觉规范。

---

## 1. 设计理念

### 核心原则
- **极简主义**：采用类似特斯拉车载 UI 的干净简洁设计哲学，最大化减少视觉干扰
- **内容优先**：让用户专注于帖子内容本身，功能服务于内容展示
- **高对比度**：清晰的视觉层次，易于阅读和导航
- **一致性**：统一的组件风格和交互模式，降低用户学习成本
- **流线型导航**：简化的导航结构，快速访问信息流、通知、消息和个人主页

### X 平台设计趋势参考
| 趋势 | 说明 | Campus Memory 应用 |
|------|------|-------------------|
| 深色模式 | 提供深色主题，节省 OLED 屏幕电量 | 支持浅色/深色双主题 |
| 视频优先 | 增强视频内容展示 | 图片/视频混合展示 |
| 实时搜索 | 输入时即时显示建议 | 搜索框实时联想 |
| 隐私增强 | 隐藏点赞列表等隐私保护 | 校友信息分级可见 |
| 内容组织 | 帖子收藏夹功能 | 收藏分类管理 |

**样式规范：**
- 卡片内边距：16px
- 头像尺寸：48px × 48px，圆形
- 头像与内容间距：12px
- 用户名：15px，Bold，主文字色
- @handle：15px，Regular，次级文字色
- 时间：15px，Regular，次级文字色
- 正文：15px，Regular，主文字色
- 互动栏图标间距：均匀分布

### 5.2 导航项 (NavItem)
```
┌─────────────────────────────┐
│  🏠  首页                    │  选中状态：Bold + 主色
├─────────────────────────────┤
│  🔍  探索                    │  默认状态：Regular + 主文字色
├─────────────────────────────┤
│  🔔  通知        ●           │  带红点提示
├─────────────────────────────┤
│  ✉️  消息                    │
├─────────────────────────────┤
│  👤  个人主页                │
└─────────────────────────────┘
```

**样式规范：**
- 高度：50px
- 内边距：12px
- 图标尺寸：26px
- 图标与文字间距：20px
- 悬停效果：背景色变为 `rgba(15, 20, 25, 0.1)`
- 圆角：9999px（胶囊形）

### 5.3 按钮 (Button)

**主要按钮（发帖按钮）：**
- 背景色：`#1D9BF0`
- 文字色：`#FFFFFF`
- 字号：17px，Bold
- 高度：52px
- 圆角：9999px
- 悬停：背景色加深 `#1A8CD8`

**次要按钮：**
- 背景色：透明
- 边框：1px solid `#CFD9DE`
- 文字色：`#0F1419`
- 悬停：背景色 `rgba(15, 20, 25, 0.1)`

### 5.4 输入框 (Input)
- 背景色：`#EFF3F4`（浅色模式）/ `#202327`（深色模式）
- 边框：无（默认）/ 2px solid `#1D9BF0`（聚焦）
- 圆角：9999px（搜索框）/ 4px（多行输入）
- 内边距：12px 16px
- 占位符色：`#536471`

### 5.5 头像 (Avatar)

|------|------|
| 帖子列表 | 48px |
| 评论 | 40px |

- 形状：圆形

## 8. 阴影规范

| 用途 | 阴影值 |

## 9. 动画规范

### 过渡时间
- 快速：150ms（悬停效果）
- 标准：200ms（状态切换）
- 慢速：300ms（页面切换）

### 缓动函数
```css
transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
```

### 常用动画
- 点赞：心形图标放大后回弹
- 加载：骨架屏闪烁动画
- 页面切换：淡入淡出

## 12. 交互设计规范

### 12.1 信息流交互
- **Tab 切换**：顶部 Tab 切换不同信息流（"推荐" / "关注" → Campus Memory 使用 "全部" / "我的圈子"）
- **下拉刷新**：下拉触发刷新动画，加载最新内容
- **无限滚动**：滚动到底部自动加载更多内容
- **回到顶部**：点击顶部导航栏标题快速回到顶部

### 12.2 帖子互动
- **点赞动画**：心形图标点击后放大 → 变色 → 回弹
- **收藏反馈**：书签图标填充动画
- **评论展开**：点击评论数展开评论区，支持折叠
- **图片预览**：点击图片全屏预览，支持左右滑动

### 12.3 搜索体验
- **实时联想**：输入时即时显示搜索建议
- **搜索历史**：显示最近搜索记录
- **热门搜索**：展示当前热门话题/圈子
- **筛选排序**：搜索结果支持按时间、热度排序

### 12.4 通知系统
- **分类展示**：互动通知、系统通知分 Tab 展示
- **未读标记**：红点/数字角标提示未读数量
- **批量操作**：支持全部标记已读

```javascript
  token: {
    colorPrimary: '#1D9BF0',
```

## 16. 状态与反馈

### 16.1 加载状态
- **骨架屏**：内容加载时显示灰色占位块，模拟内容结构
- **加载动画**：圆形旋转加载指示器，使用主色调
- **下拉刷新**：顶部出现加载指示器

### 16.2 空状态
- **无内容**：友好的插图 + 引导文案
- **无搜索结果**：提示换个关键词试试
- **无网络**：显示重试按钮

### 16.3 错误状态
- **Toast 提示**：轻量级错误提示，3秒后自动消失
- **内联错误**：表单字段下方显示红色错误文字
- **全屏错误**：严重错误显示错误页面 + 重试按钮
