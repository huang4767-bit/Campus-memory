# 好友系统 PRD

## 功能概述
好友系统支持用户之间建立社交关系，包含好友请求（带附言）、好友管理和黑名单功能。

## 功能清单

| 功能 | 优先级 | 状态 |
|------|--------|------|
| 发送好友请求（带附言） | P1 | ✅ 已完成 |
| 接受/拒绝好友请求 | P1 | ✅ 已完成 |
| 好友列表 | P1 | ✅ 已完成 |
| 删除好友 | P1 | ✅ 已完成 |
| 黑名单管理 | P1 | ✅ 已完成 |

## 业务规则

### 好友请求
- 不能添加自己为好友
- 已是好友不能重复添加
- 待处理请求不能重复发送（返回 409）
- 对方已发送请求时提示直接处理
- 支持申请附言（最多 100 字）

### 黑名单
- 拉黑后自动解除好友关系
- 拉黑后取消双方待处理的好友请求
- 被拉黑后无法发送好友请求
