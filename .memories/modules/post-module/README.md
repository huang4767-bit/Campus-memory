## 模块简介

帖子模块实现圈子内的图文发帖、帖子管理功能，支持图片上传、敏感词过滤、多种排序方式。

## 关键决策

| 决策 | 选择 | 原因 |
|------|------|------|
| 图片存储 | 本地存储 + 先上传后发帖 | 简单可靠，用户体验好 |
| 敏感词过滤 | 关键词匹配 | 练手项目，简单实现 |
| 排序方式 | 最新/热度/最新回复 | PRD 要求 |
| 热度计算 | 浏览 + 点赞×2 + 评论×3 | 综合衡量帖子热度 |
| 图片标签存储 | JSON 字段 | 简化开发，避免多表关联 |

## 相关模块

- 依赖：circles（话题圈）、users（用户）
- 被依赖：3.3 互动模块（点赞/评论/收藏）
