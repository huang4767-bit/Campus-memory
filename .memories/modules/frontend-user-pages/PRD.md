# 2.3 前端用户页面 - 产品需求文档

## 功能概述

实现用户资料相关的前端页面，完成用户信息流程闭环。

## 页面清单

| 页面 | 路由 | 功能 |
|------|------|------|
| 完善信息页 | `/profile/complete` | 新用户必填校友信息 |
| 个人主页 | `/profile` | 查看自己的资料 |
| 编辑资料页 | `/profile/edit` | 修改个人资料 |
| 他人主页 | `/user/:id` | 查看其他用户 |

## 核心流程

```
用户登录 → 检查 is_profile_complete
    ├── false → 跳转完善信息页 → 填写提交 → 首页
    └── true → 正常访问
```

## 功能要求

### 完善信息页
- 必填：真实姓名、学校、入学年份、毕业年份、班级
- 学校选择：省市联动 + 学校搜索
- 找不到学校可新增

### 个人主页
- 展示头像、昵称、简介、校友信息
- Tabs：我的帖子、我的收藏（数据待 3.x 阶段）

### 编辑资料页
- 头像上传（支持裁剪）
- 修改昵称、简介、校友信息

### 他人主页
- 展示公开信息
- 加好友按钮（功能待 4.x 阶段）
